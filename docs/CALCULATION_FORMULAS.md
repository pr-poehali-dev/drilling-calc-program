# Формулы расчетов обсадной колонны (для бурильных операций)

> **ВАЖНО**: Обсадные трубы используются как бурильные в данном калькуляторе

## 1. Расчет прочности труб

### 1.1 Давление разрыва (Burst Pressure)
```
Burst = (2 × σт × WT) / OD
```
где:
- σт - предел текучести материала, МПа
- WT - толщина стенки, мм
- OD - наружный диаметр, мм

**Основа**: API 5C3, ГОСТ Р 51906

### 1.2 Давление смятия (Collapse Pressure)
Зависит от отношения D/t:

**Для D/t ≤ 15 (толстостенные трубы)**:
```
Collapse = (2 × σт) / (D/t - 1)
```

**Для 15 < D/t ≤ 25 (среднестенные)**:
```
Collapse = σт / (0.465 × D/t - 6.775)
```

**Для D/t > 25 (тонкостенные)**:
```
Collapse = 46950000 × 6.895 / (D/t)³
```

**Основа**: API 5C3

## 2. Расчет бурения

### 2.1 Фактор плавучести
```
BF = 1 - (ρm / ρst)
```
где:
- ρm - плотность раствора, кг/м³
- ρst - плотность стали (7850 кг/м³)

### 2.2 Вес колонны в растворе
```
Wмуд = W × L × BF × g / 1000
```
где:
- W - вес трубы, кг/м
- L - глубина, м
- BF - фактор плавучести
- g = 9.81 м/с²

### 2.3 Крутящий момент

**Момент трения колонны**:
```
Tпipe = Wмуд × μ × (OD / 1000) / 2
```

**Момент на долоте (PDC)**:
```
Tбит = задается пользователем (типично 3-8 кН·м для PDC)
```

**Полный момент на поверхности**:
```
T = Tпipe + Tбит
```

где:
- μ - коэффициент трения (0.25-0.35 для обсаженного ствола, 0.35-0.45 для открытого)

**Основа**: БурсофтПроект, методика расчета torque & drag

### 2.4 Нагрузка на крюк
```
Fкрюк = Wмуд + (μ × Wмуд) + WOB
```
где:
- WOB - вес на долото (осевая нагрузка), кН

### 2.5 Максимальные обороты
```
RPMмакс = min(120, √((Tмакс × 1000) / (0.05 × OD × L / 1000)))
```

**Ограничение**: не более 120 об/мин для безопасной работы

### 2.6 Механическая скорость
```
Vмех = (RPM × Dдолота × π) / 60
```
результат в м/ч

## 3. Расчет спуска

### 3.1 Сила трения
```
Fтр = Wмуд × μобс
```
где μобс - коэффициент трения в обсаженном стволе (обычно 0.25)

### 3.2 Нагрузка при спуске
```
Fспуск = Wмуд + Fтр
```

### 3.3 Давление на забое
```
P = ρm × g × L / 1000
```
результат в МПа

### 3.4 Скорость в затрубье
```
Vзатр = Q / (π/4 × (D²скв - OD²))
```
где:
- Q - расход раствора, м³/с
- Dскв - диаметр скважины, м
- OD - наружный диаметр трубы, м

## 4. Гидравлический расчет

### 4.1 Число Рейнольдса
```
Re = (ρm × V × Dг) / μ
```
где:
- V - скорость потока, м/с
- Dг - гидравлический диаметр, м
- μ - динамическая вязкость, Па·с

### 4.2 Критическая скорость (переход в турбулентность)
```
Vкрит = (2100 × μ) / (ρm × Dг)
```

### 4.3 Потери давления в трубах (ламинарный режим)
```
ΔPтр = (32 × μ × V × L) / D²
```

### 4.4 Потери давления в затрубье
```
ΔPзатр = (ΔPтр × (Dзатр / Dтр)⁴) × 0.7
```

### 4.5 Эффективность очистки
```
η = min(100, (Vзатр / Vкрит) × 100)
```

**Критерии**:
- η < 40% - неудовлетворительная
- 40% ≤ η < 70% - удовлетворительная
- 70% ≤ η < 85% - хорошая
- η ≥ 85% - отличная

## 5. Прочность резьбовых соединений

### 5.1 Максимальный момент соединения
```
Tсоед = σт × π × OD² × 0.15 / 1000
```

Коэффициент зависит от типа резьбы:
- Buttress: 0.15
- API 8-Round: 0.12
- Premium: 0.18

### 5.2 Коэффициент запаса по моменту
```
Kз_момент = Tсоед / Tтек
```

**Критерии безопасности**:
- Kз ≥ 1.5 - безопасный режим
- 1.2 ≤ Kз < 1.5 - ограниченный запас
- Kз < 1.2 - критическая перегрузка

### 5.3 Максимальная осевая нагрузка
```
Fмакс = σт × π × (OD² - ID²) / 4
```

## 6. Профиль давлений

### 6.1 Гидростатическое давление
```
Pгидр(h) = ρm × g × h / 1000
```

### 6.2 Давление при циркуляции
```
Pцирк(h) = Pгидр(h) + ΔPтр × (h / L)
```

### 6.3 Давление в затрубье
```
Pзатр(h) = Pгидр(h) - ΔPзатр × (h / L)
```

### 6.4 Перепад давления
```
ΔP(h) = Pцирк(h) - Pзатр(h)
```

## 7. Распределение напряжений

### 7.1 Осевое напряжение
```
σос = Fкрюк / (π × (OD² - ID²) / 4)
```

### 7.2 Процент нагрузки от предела разрыва
```
%Load = (Fкрюк / Burst) × 100
```

### 7.3 Коэффициент запаса прочности
```
Kз = Burst / σос
```

## Примечания

- Все расчеты выполнены в **системе СИ** (метры, МПа, кН)
- Коэффициенты трения берутся из практики бурения в РФ
- Формулы адаптированы для использования обсадных труб в качестве бурильных
- PDC долота обычно требуют момента 3-8 кН·м в зависимости от диаметра
- Шарошечные долота: 2-5 кН·м
- Алмазные долота: 4-10 кН·м
