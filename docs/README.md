# Документация API Casing Calculator

## Обзор

Профессиональный калькулятор для расчета прочности и эксплуатационных параметров обсадных труб, используемых в качестве бурильных.

## Основные возможности

### 1. Расчет прочности
- Давление разрыва (Burst) по API 5C3
- Давление смятия (Collapse) с учетом D/t
- Коэффициенты запаса прочности

### 2. Расчет бурения
- Крутящий момент на поверхности и на долоте
- Нагрузка на крюк с учетом трения
- Осевая нагрузка и WOB
- Максимальные обороты
- **Выбор типа долота** (PDC, Шарошечное, Алмазное)
- **Настраиваемый момент на долоте**

### 3. Расчет спуска
- Вес в растворе (с фактором плавучести)
- Сила трения
- Нагрузка при спуске
- Давление на забое

### 4. Гидравлический расчет
- Потери давления в трубах и затрубье
- Критическая скорость
- Эффективность очистки скважины
- Число Рейнольдса и режим течения

### 5. Визуализация
- Профессиональные интерактивные графики (Recharts)
- Профиль давлений по глубине
- Визуализация сечения трубы
- Распределение напряжений
- Графики крутящего момента и нагрузок

### 6. Экспорт результатов
- **Экспорт в PDF** с таблицами и расчетами
- **Экспорт в Word (.docx)** с форматированием
- Включает все параметры и результаты расчетов

## База данных труб

### Российские производители (ГОСТ Р 51906)
- **ОТТМ** - Орский трубный завод
- **БТС** - Белорецкий трубный завод  
- **ТМК** - Трубная Металлургическая Компания

Каждая труба включает:
- Наружный диаметр и толщину стенки
- Марку стали и предел текучести
- Предварительно рассчитанные Burst и Collapse
- Тип резьбового соединения
- Максимальную растягивающую нагрузку

### API стандарты
Поддерживаемые марки стали:
- H-40, J-55, K-55
- N-80, L-80
- C-90, P-110

## Единицы измерения

**Система СИ (метрическая)**:
- Длина: метры (м)
- Диаметр: миллиметры (мм)
- Вес: килограммы на метр (кг/м)
- Давление: МегаПаскали (МПа)
- Сила/Нагрузка: килоНьютоны (кН)
- Момент: килоНьютон-метры (кН·м)
- Плотность: кг/м³ или г/см³
- Расход: литры в секунду (л/с)
- Вязкость: сантипуазы (сП)

## Коэффициенты трения

**Рекомендуемые значения**:
- Открытый ствол: 0.35 - 0.45
- Обсаженный ствол: 0.25 - 0.30
- С применением смазки: 0.18 - 0.25

## Типы долот и моменты

### PDC долота
- Рекомендуемый момент: **3.0 - 8.0 кН·м**
- Применение: мягкие и средние породы
- Преимущества: высокая скорость проходки

### Шарошечные долота
- Рекомендуемый момент: **2.0 - 5.0 кН·м**
- Применение: твердые породы
- Преимущества: надежность, долговечность

### Алмазные долота
- Рекомендуемый момент: **4.0 - 10.0 кН·м**
- Применение: очень твердые породы
- Преимущества: максимальная износостойкость

## Критерии безопасности

### Коэффициент запаса прочности
- **Kз ≥ 1.5** ✅ - Безопасный режим работы
- **1.2 ≤ Kз < 1.5** ⚠️ - Ограниченный запас
- **Kз < 1.2** ❌ - Критическая перегрузка

### Эффективность очистки
- **η < 40%** - Неудовлетворительная
- **40% ≤ η < 70%** - Удовлетворительная
- **70% ≤ η < 85%** - Хорошая
- **η ≥ 85%** - Отличная

### Использование нагрузки
- **< 80%** ✅ - Безопасная зона
- **80-100%** ⚠️ - Повышенное внимание
- **> 100%** ❌ - Перегрузка
